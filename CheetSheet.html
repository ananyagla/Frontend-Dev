Filters

Use filters like grayscale(100%) or blur(5px) to style images.

Media Queries

Example: @media (max-width: 700px) { ... } adjusts styles for smaller screens.

Responsive Design

Use flexbox or grid layouts with relative units like vw and %.

Typography

Scale text with font-size: clamp(1rem, 2vw, 2rem); for better readability.
