CSS Box Model Exploration

Understanding how margin, border, padding, and content interact.

Content-Box Example

content-box (default)

Width and height apply only to the content area. Padding and border add extra space around it, increasing the total size.

Border-Box Example

border-box

Width and height include content, padding, and border. This makes sizing easier and prevents the box from expanding unexpectedly.
